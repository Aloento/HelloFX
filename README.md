# HelloFX
记录了我学习JavaFX的历程，也可以作为你学习的参考资料  
内有大量中文注释，尽可能做到每一行都有

## Maven
首先，这是一个简单的Maven项目  
在任何支持Maven的IDE中打开即可使用

## JavaFX
其次，这是一个简单的JavaFX项目  
在Java11中被移除，交社区维护，改名叫OpenJFX  
使用Java15，在JVM启动参数中添加  
` --module-path "你的OpenJFX目录\javafx-sdk-15.0.1\lib" --add-modules javafx.controls,javafx.fxml `  
即可正常启动程序
